# rainbow-fusion-repo
RainbowHype's collection of projects relating to Clickteam Fusion. This can include: Extensions, Extension ports (official or unofficial), shaders, and possibly even MFAs (mostly examples rather than serious game projects).

For individual releases of any projects in here, check the releases page. (Note that they may not be updated for every hotfix made, unless it's the last hotfix for a long while.)

Enjoy your time here! :D
